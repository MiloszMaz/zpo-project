# Pobieranie kursu walut z API NBP

Dane pobierane sa z oficjalnego API NBP za pomoc¹ przycisku "Pobierz".

Dane sa wyswietlane w tabeli po rozpoczeciu programu, a przycisk "Pobierz" odswieza tabele.

Wsrod kursu walut mozna filtrowac po kodzie, po prawej stronie.

Po lewej stronie mozna przeliczyc kwote z jednej waluty na druga za pomoca kalkulatora.
